# My Learning Lab

Welcome to *My Learning Lab*, a growing multi‑page learning project.  
This repository is designed to evolve into a structured knowledge hub, covering diverse subjects step by step.

## 🌱 Vision
The goal of this project is to create a space where ideas, experiments, and notes can grow into a complete learning lab.  
Each page will explore a different topic — from coding practices and debugging notes to creative prompts and study materials.

## 📂 Structure
- *index.html* → Entry point of the site    
- *style.css* → Unified design system for consistent look and feel  
- *script.js* → Interactive features and demos  
- *notes/* → General study notes and references  
- *bugs/* → Debugging logs and fixes  
- *git/* → Git workflows and command examples  
- *python/* → Practice codes and algorithms  
- *prompts/* → AI prompts and creative ideas  

## 🚀 Roadmap
- Expand into multiple pages covering different subjects  
- Add detailed notes and examples in each folder  
- Deploy via GitHub Pages for easy access  
- Continuously refine design and interactivity  

## 🤝 Contribution
This is a personal learning lab, but ideas and improvements are always welcome.  
Feel free to fork, explore, and suggest enhancements.
